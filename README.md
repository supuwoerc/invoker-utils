# @supuwoerc/utils

:lollipop: TypeScript开发日常使用的一些方法

## 安装

这个项目使用 [node](http://nodejs.org) 和 [npm](https://npmjs.com)。请确保你本地安装了它们。

### npm

`npm install @supuwoerc/utils --save`
### yarn

`yarn add @supuwoerc/utils`

### pnpm

`pnpm add @supuwoerc/utils`

## 示例

```typescript
import { getCSSPath } from "@supuwoerc/utils"
console.log(getCSSPath(html-element))
```
